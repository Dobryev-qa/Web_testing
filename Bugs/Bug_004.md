# Bug Report – Bug_004

## Summary
Registration allows passwords longer than the maximum allowed length (20 characters).

---

## Project Information
- Project: WebShop Demo OpenCart
- URL: https://demo.opencart.com/en-gb?route=account/register
- Component: Account – Registration / Password Validation
- Platform: Web

---

## Environment
- OS: macOS Tahoe 26.0.1
- Browser: Google Chrome 142.0.7444.60

---

## Reporter
Dmytro Dobryev

## Date
25.10.2025

---

## Preconditions
- Homepage is opened
- User is not logged in

---

## Steps to Reproduce
1. Open the Registration page
2. Fill all required fields with valid data
3. Enter a password longer than 20 characters in the **Password** field
4. Click the **Continue** button

---

## Expected Result
- Password length validation error is displayed
- Registration is not completed

---

## Actual Result
- No validation error is displayed
- User is successfully registered with a password longer than 20 characters

---

## Severity
Critical

## Priority
High
